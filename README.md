# FDF
Water quality field data formatting tool that assists in NSW DPI Water imports to WISKI water quality database system.

## Overview
FDF is a graphical utility to assist water quality field officers prepare the data collected for import to the NSW DPI water quality data management system. It is written in Python with a PyQt GUI.

## Dependencies
The FDF utility relies on the following third-party packages in order to run:
* PyQT4
* PyYAML
* dateutil
